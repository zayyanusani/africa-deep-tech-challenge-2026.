# Africa Deep Tech Challenge 2026

## Team Information

**Team Name:** zayyanu-sani-team

**Lead:** Zayyanu Sani

**GitHub:** https://github.com/zayyanusani

## Project Title

Offline Coding Assistant for African Students

## Problem Statement

Many students and beginner developers in Africa have limited internet access and cannot continuously use cloud-based AI coding assistants.

## Solution

An offline coding assistant powered by DeepSeek-Coder running locally on affordable laptops with 8GB RAM.

## Domain

Coding Assistants

## Languages Supported

- English
- Hausa

## Hardware Target

- 8GB RAM
- Integrated Graphics
- Windows/Linux Laptop

## Model

DeepSeek-Coder (GGUF Quantized)

## Optimizations

- GGUF format
- Quantization (Q4)
- llama.cpp inference
- Reduced memory footprint

## Features

- Code generation
- Code explanation
- Debugging assistance
- Programming tutoring

## Test Prompt 1

Write a Python factorial function.

### Result

Successful code generation.

## Test Prompt 2

Explain JavaScript loops in Hausa.

### Result

Successful Hausa explanation.

## Impact

The project provides low-cost AI coding assistance for students and developers across Africa without requiring constant internet access.
